Version de Unity: 6000.2.6f2
Fait conjointement avec le cours IMN502 - Environnements immersifs (soit VR/AR)

Le jeu est conçu à la base pour être jouer avec des contrôles VR (principalement avec Meta Quest). Je conseille fortement si vous en possédez de tester le jeu avec. Il suffit d'avoir Meta Link (pour un Meta Quest) et le casque brancher à l'ordinateur. Le build se trouve dans le dossier /BuildVRWindows du projet.

Sinon, il est aussi possible de jouer clavier/souris. Les contrôles pour la manette ont aussi été mappé et il est tout à fait possible de jouer avec la manette, mais il se peut que certains contrôles n'agissent pas correctement (exemple: la vitesse de rotation des objets). Il est donc préférable d'utiliser clavier/souris. Le build se trouve dans le dossier /BuildWindows du projet.

Contrôles:
WASD / Joystick gauche -> Bouger
Déplacement souris / Joystick droit -> Regarder, ou tourner/monter les objets si contrôle utiliser.
Shift / Bouton joystick droit -> S'accroupir
Q / Bouton gauche -> Prendre dans la main gauche
E / Bouton droit -> Prendre dans la main droite
Clic gauche / Gâchette gauche -> Si main gauche tiens un objet, tourne l'objet de la main gauche. Sinon permet de monter/descendre l'objet de la main droite.
Clic droit / Gâchette droite -> Si main droite tiens un objet, tourne l'objet de la main droite. Sinon permet de monter/descendre l'objet de la main gauche.

Le temps imparti commence dès que le jeu est ouvert et fini après 10 minutes (modifiable sur l'inspecteur de l'objet GameManager sur Unity) lorsque le tracteur se rend au bout de la zone jouable. Après quoi une fenêtre apparaitra pour indiquer le score de la partie.
