# Jeu VR Cabane à sucre
Projet solo de jeu VR pour des cours de la session d'automne 2025 à l'Université de Sherbrooke.
Fait sur Unity avec les packages VR de Meta Quest et des assets gratuits.

Vidéo démo :
https://github.com/user-attachments/assets/97bdb487-1729-4f1d-bf2d-1396b49f80bc

Il est recommandé de tester avec un casque VR (testé avec Meta Quest et Meta Link), mais il est aussi possible d'utiliser clavier/souris.

Contrôles pour clavier/souris :  
WASD -> Bouger  
Déplacement souris -> Regarder, ou tourner/monter les objets si contrôle utiliser.  
Shift -> S'accroupir  
Q -> Prendre dans la main gauche  
E -> Prendre dans la main droite  
Clic gauche -> Si main gauche tiens un objet, tourne l'objet de la main gauche. Sinon permet de monter/descendre l'objet de la main droite.  
Clic droit -> Si main droite tiens un objet, tourne l'objet de la main droite. Sinon permet de monter/descendre l'objet de la main gauche.  

Le temps imparti commence dès que le jeu est ouvert et fini après 10 minutes lorsque le tracteur se rend au bout de la zone jouable.
